# 🚀 Bootcamp FizzBuzz

A simple JavaScript program that iterates through numbers from 1 to 100 and outputs "Fizz", "Buzz", or "FizzBuzz" based on conditional logic.

## Objectives
- Practice using `for` loops.
- Practice using `if / else if / else` conditional statements.
- Use the modulo operator `%` to check for multiples.

## 🛠️ How It Works
- Outputs **"Fizz"** for multiples of 3.
- Outputs **"Buzz"** for multiples of 5.
- Outputs **"FizzBuzz"** for numbers that are multiples of both 3 and 5.
- Prints the number itself if none of the conditions are met.
